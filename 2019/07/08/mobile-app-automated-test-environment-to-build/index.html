开发工具 eclipse + ADT + SDK

  1. 下载 Eclipse for Android Developers

  2. 为 Eclipse 安装 ADT 插件 (推荐在线安装，我这里使用的是在线安装)
    a. 启动 Eclipse，点击 Help 菜单 -> Install New Software… ?，点击弹出对话框中的 Add… 按钮

    b. Name 输入 ADT Plugin ，Location 输入 http://dl-ssl.google.com/android/eclipse/，点击 OK

    c. 在弹出的对话框选择要安装的工具，然后下一步下一步重启就可以了

    d. 安装好 ADT 插件后，Eclipse 的工具栏会出现一排 Android 的图标

  3. 在 SDK Manager 中安装需要的包
    a. 设置镜像代理，打开 SDK Manager -> Tools ，安装如下设置 （HTTP Proxy server: mirrors.zzu.edu.cn），点击 close

    b. 打开 Packages -> Reload,就会出现如下这些包

    c. 选择需要安装的包即可

    这个命令可列出该应用在系统中所存在的位置 比如： where adb.exe

    查看端口被哪个进程占用
